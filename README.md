# Email-Sender
A Email Sender made in Django 3.1.

## Como usar
```
pip install django
```
```
C:\path_to_project> python manage.py runserver
```
Acesse localhost:8000 ou 127.0.0.1:8000 no seu navegador e desfrute do projeto.
